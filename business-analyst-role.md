# Business Analyst Role - Orange Doorhouse Inc

## 🎯 Role Overview

**Position:** Business Analyst / Product Operations
**Team:** Business Analysis (BA)
**Reports to:** Head Engineer
**Status:** To be hired via Orange HR Bot

## 📋 Responsibilities

### Linear Workspace Management
- Own and optimize Linear workspace structure
- Create and maintain teams: ENG, DSN, DOC, QA, PROD, BA
- Define and improve workflows and processes
- Set up roadmaps, cycles, and sprints
- Configure labels, priorities, and custom fields

### Process Optimization
- Streamline engineering workflow (GitHub ↔ Linear sync)
- Implement Agile/Scrum practices
- Define ticket creation and review processes
- Optimize team velocity and capacity planning
- Create reporting and analytics dashboards

### Stakeholder Communication
- Generate weekly/monthly progress reports
- Track and communicate team velocity
- Create executive summaries for leadership
- Facilitate sprint planning and retrospectives
- Manage stakeholder expectations

### Tool Integration
- Maintain Linear ↔ HR Dashboard integration
- Set up additional tool integrations (GitHub, CI/CD, monitoring)
- Create automation workflows
- Ensure data consistency across systems
